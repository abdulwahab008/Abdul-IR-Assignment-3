# AI Disclosure Screenshots

Place your screenshots of AI tool conversations in this folder.

## Naming Convention
- `figure_a1_preprocessing.png` - Screenshot for preprocessing code
- `figure_a2_bm25.png` - Screenshot for BM25 implementation
- etc.

## Required Information for Each Screenshot

For each screenshot, document in the technical report:
1. The prompt you used
2. Which file/lines were affected
3. What modifications you made to the AI output

## Example

**Figure A1:** Screenshot showing the prompt used to generate text preprocessing code.

- **Prompt:** "Create a Python class for text preprocessing..."
- **Affected Files:** `src/preprocessing.py` (lines 27-180)
- **Modifications:** Added custom stopword support, error handling for NLTK downloads
